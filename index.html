1. Quick Tips & Mindset Hacks▶
  • AO Breakdown: AO1=30%, AO2=35%, AO3=35%
  • Time Slots: A:30min | B:45min | C:45min
  • Formula Cheat: Keep list of 5 key formulas on sticky note (PED, YED, multiplier)
  • Diagram Hack: Draw curves lightly then darken; label axes before writing text
  • Common Pitfalls: forgetting definitions, skipping evaluation, unlabeled diagrams
  • Exam Routine: Sleep well, light review morning-of, protein breakfast
  • Mindset: Visualize success—5 deep breaths before each section; positive affirmations
  • Quick Drill: 20-min mini essay: random micro topic → intro+diagram + 2 eval points
  • Last Check: 5-min glance at command words and syllabus terms before writing
2. Section A: Data Response▶
  1. Read & Annotate: underline command, data trends, context clues
  2. Define Terms: 1-2 lines using exact syllabus wording
  3. Apply Data: quote figures ("sales rose 15% from X to Y"), calculate % change
  4. Diagram/Table: label shifts, equilibrium, area of welfare change
  5. Analysis: cause→effect chains: "Because price fell…, therefore demand…"
  6. Evaluation: 3 bullet points: data limitations, ceteris paribus, alternative factors
  7. Link Back: summarise net effect on welfare, equity, efficiency

Model Evaluation Bank:

  • Data is a snapshot; may not show long-run trends
  • Elasticity may change over different price ranges
  • Other market forces (taxes, subsidies) could alter outcomes
3. Section B: Micro Essays▶
  • Essay Structure: Intro → Diagram → Analysis → Evaluation → Conclusion
  • Intro Starters: “This essay examines…”, “Economists define… as…”
  • Key Diagrams: tax incidence, subsidy, PED curves, consumer/producer surplus
  • Analysis Tips: use real-world examples (e.g., Pakistan fuel price cap)
  • Evaluation Bank: ceteris paribus caveat; equity vs efficiency; admin costs; black market risk
  • Linking Phrases: “Furthermore…”, “However…”, “Consequently…”
  • Conclusion: succinctly weigh pros vs cons and state overall effect
4. Section C: Macro Essays▶
  • Key Chains: “↑Govt spending→↑AD→↑Y by multiplier (multiplier=1/(1-MPC))”
  • Contextual Points: political lag in Pakistan; import leakages; informal sector
  • Diagrams: AD/AS shifts, Keynesian LRAS, multiplier diagram
  • Evaluation Bank: time lags; crowding out; debt sustainability; liquidity trap
  • Global Links: impact of global commodity prices, exchange rate shocks
  • Conclusion: compare policies and recommend most effective in given context
