  • AO1:30%, AO2:35%, AO3:35% – keep this ratio balanced.

  • Formula Cheat Sheet: PED, YED, multiplier, MPC, MPS.

Section A: Data Response
  • Read & annotate question: underline keywords and data.
  • Define 2–3 terms using syllabus definitions.
  • Quote figures & calculate % changes.
  • Draw diagram: label shifts, equilibrium, areas.
  • Analysis: “Because… therefore… leading to…” chains.
  • Evaluation: 3 bullet points on data limits & assumptions.
  • Link back: summarize welfare & efficiency impact.
Section B: Micro Essays
  • Structure: Intro → Diagram → Analysis → Evaluation → Conclusion.
  • Intro starters: “This essay examines…”, “Economists define…”.
  • Key diagrams: tax, subsidy, PED, producer/consumer surplus.
  • Analysis tip: integrate real-world examples (e.g., subsidy removal).
  • Eval phrases: “However…”, “Alternatively…”, “A possible drawback…”.
  • Conclusion: weigh pros & cons, deliver final judgment.
Section C: Macro Essays
  • Key chain: ↑G → ↑AD → ↑Y by multiplier (1/(1-MPC)).
  • Context: political delays, import leakages, informal sector.
  • Diagrams: AD/AS shifts, Keynesian LRAS, multiplier graph.
  • Eval bank: time lags, crowding out, debt sustainability.
  • Global factors: commodity price shocks, exchange rate volatility.
  • Conclusion: compare policies & recommend based on context.
